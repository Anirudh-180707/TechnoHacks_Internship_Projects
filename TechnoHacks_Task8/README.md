# Task 8 – To-Do List Application

## Overview

This project is a simple and interactive To-Do List application developed as part of the **TechnoHacks Web Development Internship**. The application enables users to add new tasks, mark tasks as completed, and remove tasks through a clean and responsive user interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive and user-friendly design
- Interactive interface using JavaScript
- Simple and clean layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Task8_To_Do_List/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task8_To_Do_List** folder.
3. Open `index.html` in any modern web browser.
4. Enter a task in the input field and click **Add**.
5. Mark tasks as completed or delete them as needed.

## Components

- **Task Input** – Allows users to enter new tasks.
- **Add Button** – Adds a new task to the list.
- **Task List** – Displays all added tasks.
- **Complete Button** – Marks a task as completed.
- **Delete Button** – Removes a task from the list.
