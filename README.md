# TechnoHacks Web Development Internship

This repository contains the projects completed during my **TechnoHacks Web Development Internship**. Each task focuses on strengthening fundamental front-end web development skills using **HTML**, **CSS**, and **JavaScript**.

## Tasks Included

* Task 1 – Personal Portfolio Website
* Task 2 – Responsive Navigation Bar
* Task 3 – Landing Page
* Task 4 – Quiz Application
* Task 5 – CSS Animations
* Task 6 – Responsive Grid Layout
* Task 7 – Contact Form
* Task 8 – To-Do List Application
* Task 9 – Product Card
* Task 10 – Blog Page
* Task 11 – Weather App Interface
* Task 12 – E-commerce Product Page

Each task is organized in its own folder and includes a dedicated README containing:

* Project Overview
* Features
* Technologies Used
* Project Structure
* How to Run
* Screenshots (if applicable)

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

The objective of this internship is to gain hands-on experience in front-end web development by building responsive, interactive, and user-friendly web pages and applications using core web technologies.
