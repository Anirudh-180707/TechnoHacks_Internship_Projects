# Task 7 – Contact Form

## Overview

This project is a responsive contact form developed as part of the **TechnoHacks Web Development Internship**. The form allows users to enter their name, email address, and message while performing basic client-side validation using JavaScript to ensure all required fields are completed before submission.

## Features

- Responsive contact form
- Name, Email, and Message input fields
- Client-side form validation
- Email format validation
- Success message after valid submission
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
Task7_Contact_Form/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task7_Contact_Form** folder.
3. Open `index.html` in any modern web browser.
4. Fill in the required fields and click **Send Message**.

## Components

- **Header** – Displays the contact form title and description.
- **Contact Form** – Collects the user's name, email address, and message.
- **Validation Script** – Checks for empty fields and validates the email format before allowing submission.
- **Success Message** – Confirms successful form submission after validation.

## Learning Outcomes

- Creating forms using HTML
- Styling forms with CSS
- Validating user input using JavaScript
- Handling form submission events
- Improving user experience with client-side validation
