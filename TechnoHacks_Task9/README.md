# Task 9 – Product Card

## Overview

This project is a responsive product card developed as part of the **TechnoHacks Web Development Internship**. The card showcases a product with its image, title, description, price, and a call-to-action button using HTML and CSS. The design is clean, responsive, and suitable for modern e-commerce websites.

## Features

- Responsive product card layout
- Product image display
- Product title and description
- Product price
- "Buy Now" call-to-action button
- Hover animation for an interactive user experience
- Clean and modern user interface

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
Task9_Product_Card/
│── index.html
│── style.css
├── images/
│   └── image.png
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task9_Product_Card** folder.
3. Ensure the product image (`image.png`) is placed inside the `images` folder.
4. Open `index.html` in any modern web browser.

## Components

- **Product Image** – Displays the featured product.
- **Product Title** – Shows the product name.
- **Product Description** – Provides a brief overview of the product.
- **Price** – Displays the product price.
- **Buy Now Button** – Represents a call-to-action for purchasing the product.
