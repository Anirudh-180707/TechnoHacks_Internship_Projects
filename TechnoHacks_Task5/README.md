# Task 5 – CSS Animation

## Overview

This project is a CSS Animation Showcase developed as part of the **TechnoHacks Web Development Internship**. It demonstrates the use of various CSS animations and transitions to create engaging and interactive user interface elements using only HTML and CSS.

## Features

* Animated page title with fade-in effect
* Bouncing ball animation
* Rotating star animation
* Interactive card with hover zoom effect
* Animated button with hover transition
* Responsive design for different screen sizes

## Technologies Used

* HTML5
* CSS3

## Project Structure

```text
Task5_CSS_Animation/
│── index.html
│── style.css
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task5_CSS_Animation** folder.
3. Open `index.html` in any modern web browser.
4. Interact with the webpage by hovering over the card and button to view the animations.

## Animations Included

* **Fade-In Animation** – Applied to the page title.
* **Bounce Animation** – Creates a continuous bouncing effect for the ball.
* **Rotate Animation** – Continuously rotates the star element.
* **Hover Scale Effect** – Enlarges the card when hovered.
* **Button Hover Effect** – Changes appearance and lifts the button on hover.

## Learning Outcomes

* Creating animations using `@keyframes`
* Applying CSS transitions and transforms
* Designing interactive user interface elements
* Building responsive webpages with CSS
* Enhancing user experience through animations
