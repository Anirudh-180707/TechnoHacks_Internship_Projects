# Task 2 – Responsive Navigation Bar

## Overview

This project demonstrates a responsive navigation bar created as part of the **TechnoHacks Web Development Internship**. The navigation is designed to provide a smooth and user-friendly experience across desktop and mobile devices using HTML, CSS, and JavaScript.

## Features

* Responsive navigation bar
* Mobile-friendly layout
* Interactive hover effects
* Responsive menu functionality
* Multi-page navigation
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
Task2_Responsive_Navigation_Bar/
│── index.html
│── about.html
│── contact.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task2_Responsive_Navigation_Bar** folder.
3. Open `index.html` in any modern web browser.

## Pages & Files

* **index.html** – Main page containing the responsive navigation bar.
* **about.html** – Demonstrates navigation between website pages.
* **contact.html** – Contact page linked through the navigation menu.
* **style.css** – Styles the navigation bar, layout, responsiveness, and hover effects.
* **script.js** – Handles interactive navigation behavior for improved user experience.

## Learning Outcomes

* Creating responsive navigation using HTML and CSS
* Implementing media queries for different screen sizes
* Adding interactivity with JavaScript
* Building reusable navigation across multiple web pages

## Future Improvements

* Add active navigation highlighting.

* Enhance accessibility with keyboard navigation support.
