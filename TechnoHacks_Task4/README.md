# Task 4 – Simple Quiz Application

## Overview

This project is a simple and interactive quiz application developed as part of the **TechnoHacks Web Development Internship**. The application allows users to answer multiple-choice questions, calculates the final score, and displays the result upon quiz completion.

## Features

* Interactive multiple-choice quiz
* Score calculation
* Result display after completing the quiz
* Restart quiz functionality
* Responsive and user-friendly interface
* Hover and selection effects for answer options

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
Task4_Quiz_Application/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task4_Quiz_Application** folder.
3. Open `index.html` in any modern web browser.
4. Select an answer for each question and click **Next**.
5. View your final score and restart the quiz if desired.

## Files

* **index.html** – Defines the quiz structure and user interface.
* **style.css** – Styles the quiz layout, buttons, and responsive design.
* **script.js** – Implements quiz logic, answer validation, score calculation, and restart functionality.
