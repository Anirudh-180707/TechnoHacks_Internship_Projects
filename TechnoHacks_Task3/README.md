# Task 3 – Landing Page

## Overview

This project is a responsive landing page developed as part of the **TechnoHacks Web Development Internship**. The landing page is designed for a fictional online learning platform named **StudyFlow**, showcasing a modern layout, engaging content sections, and clear call-to-action buttons using HTML and CSS.

## Features

* Responsive landing page design
* Modern and clean user interface
* Hero section with a call-to-action button
* Features section highlighting key services
* About section
* Responsive navigation bar
* Mobile-friendly layout using CSS media queries

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
Task3_Landing_Page/
│── index.html
│── style.css
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task3_Landing_Page** folder.
3. Open `index.html` in any modern web browser.

## Sections

* **Hero Section** – Introduces the fictional product with a headline and call-to-action button.
* **Features** – Highlights the platform's main features.
* **About** – Provides a brief description of the service.
* **Call-to-Action** – Encourages users to get started.
* **Footer** – Displays copyright information.

## Learning Outcomes

* Creating structured web pages using semantic HTML
* Designing responsive layouts with CSS
* Using Flexbox and CSS Grid for page organization
* Building attractive landing pages with modern UI principles
