# Task 6 – Responsive Grid Layout

## Overview

This project is a responsive grid layout developed as part of the **TechnoHacks Web Development Internship**. It demonstrates the use of **CSS Grid** to create a flexible and responsive layout for displaying content cards that automatically adjust to different screen sizes.

## Features

- Responsive grid layout using CSS Grid
- Six responsive content cards
- Modern and clean user interface
- Hover effects on cards
- Mobile-friendly design using media queries
- Simple and easy-to-understand layout

## Technologies Used

- HTML5
- CSS3

## Project Structure

```text
Task6_Responsive_Grid_Layout/
│── index.html
│── style.css
└── README.md
```

## How to Run

1. Clone or download this repository.
2. Open the **Task6_Responsive_Grid_Layout** folder.
3. Open `index.html` in any modern web browser.

## Components

- **Header** – Displays the project title and a brief description.
- **Grid Layout** – A responsive grid containing six content cards.
- **Content Cards** – Display sample web development topics with hover effects.
- **Footer** – Displays copyright information.

## Learning Outcomes

- Creating responsive layouts using CSS Grid
- Building reusable content cards
- Implementing hover effects with CSS
- Using media queries for responsive web design
- Designing clean and user-friendly interfaces

## Future Improvements

- Add images or icons to each card.
- Include animations for card transitions.
- Add filtering or sorting functionality using JavaScript.
